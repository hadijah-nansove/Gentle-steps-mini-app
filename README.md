Gentle Steps
Your calm, private mental wellness space.
README 

What is Gentle Steps?
It's designed to feel like a quiet, safe corner of your phone or laptop where you can check in with yourself, breathe, write, and find calm — without any of the noise of a typical wellness app.

Features
Feature	Description
🌈 Mood Check-In	Log how you're feeling with emoji moods + an optional private note
🌬️ Breathing Exercises	Guided breathing with three techniques: 4-7-8, Box Breathing, and Deep Belly
📖 Private Journal	Write freely with optional prompts. Entries are saved only on your device
🌿 5-4-3-2-1 Grounding	A tap-through sensory grounding exercise for moments of anxiety or panic
🌙 Sleep Tracker	Log your bedtime, wake time, and sleep quality to build awareness of your rest patterns
✨ Calming Stories	Short, peaceful read-along stories to help you wind down
💬 Daily Affirmations	Rotating affirmations to gently encourage and ground you
🎨 Themes	7 colour themes (Teal, Lavender, Rose, Sage, Sunset, Ocean, Midnight)
📤 Data Export	Export all your logs as a JSON file for personal backup
🗑️ Clear Data	Permanently wipe all local data with one tap — full control, always


Privacy
All data — moods, journal entries, sleep logs — is stored only in your browser's localStorage. Nothing is sent to any server. The app works fully offline via a built-in Service Worker.

No accounts. No tracking. No servers. No ads. Your words are yours alone.


How to Use
1.	Open gentle_steps.html in any modern web browser (Chrome, Firefox, Safari, Edge).
2.	No installation, no login, no internet needed.
3.	Navigate between sections using the top navigation bar (or the hamburger menu on mobile).
4.	Your data is automatically saved as you go.
